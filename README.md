# f1_project

A clean and functional web application for managing Formula 1 drivers and teams. Developed using Laravel, this project adheres to MVC design patterns and professional web development standards.

## About the Project
This system provides an intuitive interface for managing F1 driver statistics. It allows administrators to maintain the database while providing a seamless viewing experience for regular users.

## Key Features
* **Full CRUD:** Create, Read, Update, and Soft Delete functionality for driver records.
* **Role-Based Access Control:** Separate permissions for Admins (full access) and regular users (view-only).
* **Localization:** Automatic language detection based on the user's browser settings. Now available: Latvian, English.
* **Data Integrity:** Relational database structure (3NF) linking drivers to teams.
* **Security:** CSRF protection, password hashing (bcrypt), and server-side input validation.

## 🏗 Tech Stack
* **Backend:** PHP 8, Laravel Framework.
* **Database:** MySQL.
* **Frontend:** Blade, CSS, HTML.
* **Compliance:** Valid HTML5, UTF-8 encoding, and Clean URLs.
